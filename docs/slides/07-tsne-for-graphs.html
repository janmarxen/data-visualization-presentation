How is t-SNE Used for Graph Layouts?

1️⃣ Graph Representation

Convert graph nodes into high-dimensional feature vectors using:

  • Node embeddings (e.g., Node2Vec, DeepWalk)
  • Adjacency matrix representations
  • Feature vectors (node attributes, degrees, etc.)

2️⃣ Compute Similarities

Calculate pairwise distances or use graph-based distances (shortest path, random walk probabilities) to build the high-dimensional similarity matrix P.

3️⃣ Embed and Visualize

Apply t-SNE to project nodes into 2D space, where:

  • Nearby nodes → Similar/connected in original graph
  • Clusters → Communities or densely connected subgraphs
  • Outliers → Loosely connected or anomalous nodes
