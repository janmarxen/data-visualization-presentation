What is t-SNE? Mathematical Foundation

Core Idea

t-Distributed Stochastic Neighbor Embedding (t-SNE) converts distances into probability distributions and minimizes their divergence.

1️⃣ High-Dimensional Similarities (Gaussian)

For each pair of points i and j, compute conditional probability:

pj|i = exp(-||xi - xj||² / 2σi²) / Σk≠i exp(-||xi - xk||² / 2σi²)

where σi is determined by perplexity (effective number of neighbors)

2️⃣ Low-Dimensional Similarities (Student's t)

In the embedded space (2D/3D), use heavy-tailed t-distribution:

qij = (1 + ||yi - yj||²)-1 / Σk≠l (1 + ||yk - yl||²)-1
